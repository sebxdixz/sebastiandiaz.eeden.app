## 1. Agon.cl
**Tipo:** Full-Stack Web Platform  
**Rol:** Desarrollador Principal (Sistemas, Backend, Frontend, Diseño)

### Descripción
Plataforma web completa desarrollada desde cero. Responsable de:
- Arquitectura del sistema y decisiones tecnológicas
- Backend: APIs REST, base de datos, optimización
- Frontend: Interfaz responsive, UX/UI minimalista
- Diseño visual: estética limpia y moderna

### Stack Técnico
- React + TypeScript (frontend)
- Node.js + Express (backend)
- PostgreSQL (base de datos)
- CSS3 + Responsive Design
- REST APIs

### Logros
- Sistema robusto y escalable
- Performance optimizado
- Diseño minimalista coherente
- Experiencia de usuario intuitiva

## 🎯 Énfasis en Agentes IA

### ¿Por qué Agentes IA?

En el portafolio se enfatiza especialmente **JOBER** y **Orthogonal** porque representan:

1. **Innovación Técnica**
   - Uso avanzado de LLMs
   - Arquitectura multi-agent
   - Reasoning y planning automático

2. **Valor Comercial**
   - Automatización de procesos
   - Reducción de tiempo en tareas
   - Escalabilidad sin límites

3. **Diferenciador**
   - No todos pueden construir agentes IA
   - Requiere entendimiento profundo de LLMs
   - Combinación de múltiples disciplinas

### Badge de AI Agent

En el portafolio, los proyectos con tecnología de agentes IA mostrarán un badge **🤖 AI Agent** para destacarlos visualmente.
